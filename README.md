# JavaFlickrGUI

Java application that grabs images from flickr.com and show them in GUI.

I created this application to learn:
* Basic Java syntax
* Swing
* Java Threads
* Flickr API

## How to run application
```
cd dist/
java -jar JavaFlickrGUI.jar
```
